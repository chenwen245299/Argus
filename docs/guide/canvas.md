# Canvas

The canvas is a visual, node-based space for mapping how papers relate and how a field
developed — perfect for literature reviews, building a roadmap of a field, or planning a
survey.

<Media src="/media/canvas.mp4" caption="A canvas is a great way to trace how papers and techniques evolved" />

## Building a relationship map

- Add **paper nodes** and connect related work with edges.
- Mix in **text nodes**, **shape nodes**, and **image nodes** to annotate your map.
- Keep **multiple canvases** per library for different topics or projects.

The canvas is built on [Vue Flow](https://vueflow.dev/) — pan/zoom-able, with a minimap
and controls.

## Smart assistance

- **Edge suggestions** — Argus can suggest connections between related papers.
- **Auto-layout** — tidy up a messy canvas in one click.
- **Note pinning** — pin notes to the canvas for context; hover a node to see its label.

## Export

Export any canvas as an image — **PNG**, **PDF**, or **SVG** — for slides, papers, or
shared docs.
