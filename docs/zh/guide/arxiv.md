# arXiv / bioRxiv 追踪

Argus可以替用户盯着新预印本论文 —— 抓取、过滤、分析、集中处理，让用户把时间花在读对的论文上，而不是四处找论文。

## 配置

在使用arXiv / bioRxiv追踪功能之前，需要先在设置中配置相应的参数，包括：爬取分类、感兴趣的关键词、爬取频率、爬取时间、回溯天数、是否启用AI自动分析、AI分析的模型和提示词等。配置好后，Argus会按计划从arXiv / bioRxiv抓取新论文，并放进每日收件箱。

<Media src="/media/1783338672276.png" caption="带 AI 相关性评分和一键导入的每日收件箱" />

## 每日收件箱

Argus按计划从**arXiv**和**bioRxiv**抓取新论文，放进每日**收件箱**。在这里用户可以:

- 按日期分组浏览新推荐
- 标记已读 / 未读并评分
- 把有价值的论文直接加入文献库
- 按日期范围批量删除收件箱条目

## 定时抓取

后台调度器会按用户配置的节奏抓取，并根据回溯天数补上错过的日期，即使arXiv窗口没有打开，只要Argus应用正在运行，收件箱也能保持更新。

<Media src="/media/1783338965093.png" caption="用户也可以手动触发arXiv抓取" />

## AI 相关性过滤

Argus可以自动使用AI分析收集到的arXiv论文并根据用户配置的关键词筛选出用户可能感兴趣的文章，从而让用户从找论文的体力活中解放出来。Argus会**按用户设定的阈值**过滤文章，用户可以通过该阈值调整过滤的严格程度。

<Media src="/media/arxiv-ai-filter.mp4" caption="Argus能够根据用户配置的关键词自动筛选去用户感兴趣的论文" />

在AI分析过程中，低于阈值的论文会被直接删除，用户感兴趣的论文会被打上标签并保留。AI分析最多支持10篇论文同时进行，可以极大的提高AI分析的效率。
