# 画布

画布是一个可视化的、基于节点的空间，可以用来梳理论文之间的关系与发展脉络 —— 非常适合做文献综述、建立对某个领域的发展路线图，或规划一篇survey。

## 构建关系图谱

- 添加**论文节点**，用连线连接相关工作。
- 混入**文本节点**、**形状节点**和**图片节点**来标注你的图谱。
- 每个文献库可保留**多张画布**，用于不同主题或项目。

画布基于 [Vue Flow](https://vueflow.dev/) 构建，可平移 / 缩放，带小地图和控件。

## 智能辅助

- **连线建议** —— Argus 可以为相关论文之间建议连接。
- **自动布局** —— 一键整理凌乱的画布。
- **笔记钉附** —— 把笔记钉在画布上作为上下文，鼠标悬浮在对应节点上时可以看到相应的标签。

## 导出

把任意画布导出为图片 —— **PNG**、**PDF** 或 **SVG** —— 便于放进幻灯片、论文或共享文档。
