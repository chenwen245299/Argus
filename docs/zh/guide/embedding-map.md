# 向量图谱

Argus可以把文献库的embedding投影到二维图上，让用户**看见**相关工作的聚类，发现此前可能忽略的关联。向量图谱功能通过AI随航→向量图谱打开。如下图所示，LLM相关的论文和CV相关的论文被自动分割成了两个不同的类。

<Media src="/media/embedding-map.mp4" caption="通过embedding向量可视化当前文献库文献之间的关系" />

## 它能做什么

- **聚类可视化** —— 主题相近的论文在图上会自然聚到一起，一眼看出文献库有哪些板块。
- **发现关联** —— 距离相近的论文往往内容相关，方便顺藤摸瓜找到相邻工作。
- **点击定位** —— 点击图上的点即可跳转到对应论文。

## 前置条件

向量图谱基于RAG的向量索引。使用前请先在 [RAG问答](/zh/guide/rag) 里为文献库构建embedding索引（配置好embedding模型并建库）。
