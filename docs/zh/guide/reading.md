# 阅读与笔记

Argus 首先是一个论文和图书阅读器。它的设计目标是让你在阅读、批注和写作时保持专心不被打断，并且能随时召唤AI进行辅助阅读和提问。

<Media src="media/reading.mp4" caption="标签页阅读、高亮、并排记笔记" />

## 阅读器

- **标签页 PDF 阅读器** —— 由 pdf.js 驱动，可同时打开多篇论文。支持文档内全文搜索、翻页，以及持久化阅读状态(Argus 会为每篇论文记住页码和滚动位置)。
- **电子书阅读器** —— 阅读 EPUB、MOBI、AZW3、FB2、TXT，支持章节导航和目录。

## 高亮与批注

选中文本即可高亮；高亮按论文保存在 `highlights.json` 中，始终跟随论文文件夹。

## 笔记

每篇论文可写任意多条笔记，使用富文本 Markdown 编辑器(Vditor)，支持：

- **数学公式**(KaTeX)
- **图表**(Mermaid)
- **代码**(语法高亮)

笔记可在独立窗口中打开,方便你一边读论文一边写，点击笔记标题右侧的按钮即可在独立窗口中显示，见下图：

## 全文提取

为了支撑搜索、AI分析、AI对话和RAG，Argus 通过分层流水线提取每篇论文的全文:

1. 原生 PDF 文本提取(快速路径)。
2. 对棘手 PDF 使用 `pdftotext`。
3. 对扫描件用 **OCR** 兜底 —— 在 macOS 上原生调用 **Vision** 框架，其他平台用**Tesseract**。

提取出的文本存为论文文件夹里的 `fulltext.txt`，可随时重新提取全文。

<Media src="/media/1783328013050.png" caption="通过右键菜单栏触发全文提取功能" />

## 章节划分

Argus可以把论文自动切分为逻辑章节（摘要、引言、方法……），便于在长论文中跳转，也能向 AI 提供结构化的更加简短的上下文。

<Media src="/media/1783328174631.png" caption="Argus支持自动划分章节" />

## 翻译

使用翻译功能前请先配置AI供应商，并配置翻译的默认模型，如果没有配置翻译模型，翻译功能将使用全局的默认模型。建议在配置完AI供应商后设置一个默认模型：

<Media src="/media/1783328540768.png" caption="点击⭐️设置默认模型" />


内置翻译的结果与AI对话格式一致 —— 显示所用模型、输入 / 输出 token 及费用 —— 并提供重新生成、复制等快捷操作，以及翻译历史。
