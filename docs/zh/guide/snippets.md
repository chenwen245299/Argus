# 素材库

素材库是一个独立于论文之外的收集箱，用来存放用户阅读论文时的一些摘录，以便后续使用。它拥有**自己的向量库**，可以对收集的素材做语义检索，方便用户写论文时快速查找和引用。

<Media src="/media/snippets.mp4" caption="收集素材并进行语义检索" />

## 它能做什么

- **随手收集** —— 把有价值的引文、片段、想法存进素材库。
- **语义检索** —— 基于独立的向量库，按语意检索素材，而不只是关键词匹配。
- **与论文分离** —— 素材库和文献库互不干扰,专门服务于写作引用与灵感整理。

## 存储方式

素材以文件形式存放在文献库文件夹的 `snippets/` 目录下，和论文一样是本地、可迁移的。
